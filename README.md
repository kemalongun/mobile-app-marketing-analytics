# Mobile App Marketing Analytics Project

This project focuses on analyzing user acquisition and revenue performance for a mobile subscription-based application.

The goal of the project is to evaluate revenue attribution across acquisition channels and compare the performance of paid marketing platforms using key business metrics such as installs, cost per install, total revenue, return on ad spend, and revenue per install.

## Project Scope

The analysis covers three main areas:

1. **Revenue Attribution**  
   Revenue generated by users was attributed to acquisition platforms such as Facebook, Apple Search Ads, and Organic traffic.

2. **Marketing Performance Comparison**  
   Paid acquisition channels were compared using CPI, total cost, total revenue, ROAS, and revenue per install.

3. **Dashboard-Oriented Metric Preparation**  
   App event data was processed into daily metrics suitable for business intelligence dashboards and performance monitoring.

## Key Insights

- Revenue was attributed across Facebook, Apple Search Ads, and Organic users.
- Facebook showed stronger overall efficiency in terms of ROAS.
- Apple Search Ads generated higher revenue per install, but had a higher acquisition cost.
- The results suggest that both platforms can be useful, but Facebook performed better from a cost-efficiency perspective.
